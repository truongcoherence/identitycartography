## How to Read the Columns

**Evidentiary Tier** and **Exposure Tier** — taken directly from the author's registry, not re-derived.

**Case File Status** — Drafted or Pending, per the registry. Drafted means a standalone `CASE-NNN.md` file already exists; Pending means it doesn't yet.

**Audit Category** — Bedrock, Fog, or Aether, per the classification established in the Canon Audit Findings Log. **"Not yet audited"** means the case hasn't been run through that methodology yet, not that it's assumed fine.

**Documentation / Adjudication Note** — one of two things, stated plainly:

- **"Needs transcription"** — Pending in the registry; exists only as narrative inside a domain canon document, not as a standalone file. Mechanical work, not a judgment call.
- **"Needs adjudication"** — there's an open question about the case that requires a decision (resolve an ambiguity, decide reuse weighting, locate a specificity drift, decide a domain home) rather than just typing it up. A case can need both.
- **"Drafted — confirm current status"** — already has a file per the registry; the only remaining task is checking whether anything since has changed (a correction, a closed evaluation window) that the existing file doesn't yet reflect.
