# OPERATORS

In the lexicon of Identity Cartography, the term Operators refers specifically to the structural tools, lenses, and analytical instruments used to resolve the terrain of a system. Unlike the Active Canon, which maps the underlying physics, or Field Notes, which document observational data, Operators are the mechanical apparatuses used to conduct the actual work of mapping.

———-
## Core Operators 

These are the five tools you actually use. They help you see structure, interpret signals, reduce load, detect instability, and test boundaries.
You will find them here:

* [LENS](../OPERATORS/Lens%20Prompt.md)
* [INTERPRETER](../OPERATORS/Interpreter%20Prompt.md)
* [RELIEF](../OPERATORS/Relief%20Prompt.md)
* [SEISMOGRAPH](../OPERATORS/Seismograph%20Prompt.md)
* [PROBE](../OPERATORS/PROBE%20Prompt.md)

Each Core Operator has:
* a clear definition
* a usage guide
* an IFU (Instruction for Use)
* prompts you can run
* examples of correct usage

These are the only operators that are *procedural*.

———-
## Compass 
Compass is the meta‑operator. 

Compass helps you:
* anchor context
* stabilize perspective
* maintain orientation across complexity

***Compass is an operator that helps you determine which core operator to use.***

You will find it here:
* [COMPASS](../OPERATORS/Compass%20Prompt.md)
